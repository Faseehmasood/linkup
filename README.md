LinkUp Platofrom

A professional client portal built for Pakistani freelancers. 
Manage projects, milestones, invoices, and client communication all in one place.

1) The Problem

Pakistani freelancers juggle Google Docs for proposals, 
Canva for invoices, WhatsApp for updates, and manually 
track payments via Wise and Payoneer. Nothing is in one place, 
nothing is automated.

2) The Solution

LinkUp gives every freelancer a professional portal where:
- Clients get a magic link — no password, no signup
- Milestones auto-generate invoices when marked complete
- Wise / Payoneer details appear directly on invoices
- Weekly project summaries email clients automatically
- Real-time notifications keep clients updated

3) Tech Stack

- Frontend: Next.js 15, TypeScript, Tailwind CSS, Shadcn UI, Zustand
- Backend: Next.js API Routes, MongoDB, Mongoose
- Auth: NextAuth.js — Google OAuth + Magic Link
- Real-time: Socket.io
- Email: React Email + Resend
- PDF: react-pdf
- Infrastructure: Redis, Docker, GitHub Actions CI/CD

4) Features

-  Google OAuth + Magic link auth
-  Project & milestone management
-  Client portal — limited view
-  PDF invoice generator
-  Wise / Payoneer payment details on invoice
-  Milestone → auto invoice trigger
-  Weekly client summary email
-  Real-time notifications
-  Earnings dashboard
-  Docker + CI/CD pipeline
